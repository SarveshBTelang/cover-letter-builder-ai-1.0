# Cover-Letter-Builder-AI-1.0

Automated Cover Letter Generation using LLMs in docx and pdf format using custom job description and resume inputs.

Sync and update LLM context files (Instructions, Resume, Letter Structure etc.) via Cloudflare R2 API
